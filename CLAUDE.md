# Blog

Hexo-based personal tech blog. Deployed to `lightblues.github.io/blog`.

## Quick Reference

- Theme: Butterfly
- Node: 18+
- Deploy target: `Lightblues/blog` repo (gh-pages branch)

## Common Commands

```bash
npm install          # install deps
npx hexo generate    # build
npx hexo server      # local preview (localhost:4000)
npx hexo new "title" # new post
npx hexo clean       # clear cache
```

## Coding Style

- Blog posts in `source/_posts/` as Markdown
- Images referenced via jsdelivr CDN (assets repo) or relative paths
